Même destination finale
Dans les deux cas, les données arrivent dans le Microsoft Cloud de Clair-Bois (SharePoint, Power Automate). Rien ne change côté stockage.
🛡️
Chiffrement HTTPS
Le site utilise le même protocole sécurisé (HTTPS) que Microsoft Forms. Les données sont chiffrées pendant le transport.
🚫
Le site ne stocke rien
Le site web sert uniquement d'interface de saisie. Il ne conserve aucune donnée. Tout est envoyé directement à Microsoft.
🏢
Conformité LPD / RGPD
Les données restent dans le tenant Microsoft 365 de Clair-Bois. La conformité est identique à la solution actuelle.
