3

Créer une nouvelle demande

Quand un stagiaire arrive et qu'un placement doit être ouvert
1
Passez en Vue Détaillée — cliquez sur Tous les éléments en haut à droite de la liste.
2
Cliquez sur + Ajouter un nouvel élément (bouton bleu en haut).
3
Remplissez les champs dans le formulaire :
StagiaireID · TypeDemande · Secteur · DateDebutSouhaitee · ObjectifStage · ParcoursScolaire
4
Laissez le Statut sur En attente des d… — ne le changez pas encore.
5
Cliquez sur Enregistrer.
⚠ Pré-requis : le stagiaire doit déjà exister dans la liste Stagiaire.
Si ce n'est pas le cas, créez d'abord sa fiche (voir section 6 p.3).

Confirmer un placement

Une fois validé avec le secteur concerné
1
Cliquez sur la ligne de la demande En attente pour ouvrir le panneau.
2
Remplissez les champs de confirmation :
→ PlanMetier
→ EtablissementConfirme
→ SecteurConfirme
→ TypePlace
→ DateDebutConfirmee
→ DateFinConfirmee
3
Changez le Statut → Confirmé
4
Cliquez sur Enregistrer.
Mise à jour automatique ✓
Dans l'heure qui suit, la cartographie reflète le changement — le siège passe en orange (réservé futur) ou en rouge (occupé aujourd'hui).
Modifier les dates après confirmation ?
Rouvrez la demande, corrigez les dates, Enregistrer. La cartographie se met à jour automatiquement.

Cartographie — visualiser les places

Vue en lecture seule · Accès par mot de passe · Usage interne DFIP
Accès : ouvrez le lien, entrez le mot de passe communiqué par l'équipe.
URL clairbois-dfip.github.io/clair-bois-calendrier/#carto
Code couleur des places :
Vert — place libre, aucune réservation
Orange — libre aujourd'hui, réservée à une date future
Rouge — occupée en ce moment
Survol d'un siège orange ou rouge → info-bulle avec le prénom du stagiaire et ses dates.

Lecture seule. Toute modification se fait depuis la liste SharePoint Demande.

6

Créer un nouveau stagiaire

Pré-requis avant toute demande
1
Dans le menu gauche du site SharePoint, cliquez sur Stagiaire.
2
Cliquez sur + Ajouter un nouvel élément.
3
Remplissez : Prénom Nom AVS → Enregistrer.
4
Revenez dans la liste Demande pour créer la demande associée (section 3).
La liste Stagiaire est accessible depuis le menu gauche de SharePoint :
Contenu du site → Stagiaire

Un stagiaire peut avoir plusieurs demandes dans sa vie — une par période ou par changement de statut (Stage → Formation → Mesure d'orientation…). Chaque nouvelle période = une nouvelle demande avec le même StagiaireID.
